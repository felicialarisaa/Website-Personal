Menghitung Rata-Rata dari N Bilangan

1. Mulai

2. Tampilkan "NILAI RATA-RATA MAHASISWA MACHUNG"

3. Deklarasikan variabel bilangan berkoma angka, total, ratarata

4. Deklarasikan variabel bilangan bulat jmlAngka, i

5. Input jmlAngka // Jumlah mahasiswa

6. Inisialisasi total menjadi 0

7. Untuk i dari 1 sampai jmlAngka lakukan

8. Tampilkan "Masukan Nilai Mahasiswa ke", i

9. Input angka

10. Total <- total + angka

11. Akhir Untuk

12. Hitung ratarata = total / jmlAngka

13. Tampilkan "NILAI RATA-RATA MAHASISWA MACHUNG ADALAH:", ratarata

14. Selesai
