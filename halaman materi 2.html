Menentukan Bilangan Terbesar dari 3 Bilangan

1. Mulai

2. Deklarasikan variabel bilangan sebagai integer

3. Tampilkan "masukkan bilangan pertama" dan baca input bilangan pertama

4. Tampilkan "masukkan bilangan kedua" dan baca input bilangan kedua

5. Tampilkan "masukkan bilangan ketiga" dan baca input bilangan ketiga

6. Pilih berdasarkan bilangan tertinggi

Jika bilangan 1 > 2, 3 maka tampilkan "bilangan pertama adalah yang terbesar"

Jika bilangan 2 > 1, 3 maka tampilkan "bilangan kedua adalah yang terbesar"

Jika bilangan 3 > 1, 3 maka tampilkan "bilangan ketiga adalah yang terbesar"

7. Selesai
